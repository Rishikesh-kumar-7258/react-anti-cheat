Presets

Pre-configured restriction bundles for common use cases. Use them as-is or extend them with additional restrictions.

import { presets } from '@rishikesh7258/react-proctor';

presets.exam

Maximum restrictions — for timed exams and high-stakes assessments.

Enabled Restrictions

disableCopy, disablePaste, disableCut, disableTextSelection, disableRightClick, detectTabSwitch, detectDevTools, enforceFullscreen, disableKeyboardShortcuts, disableDragDrop, disablePrintScreen 

{
  disableCopy: { enabled: true },
  disablePaste: { enabled: true },
  disableCut: { enabled: true },
  disableTextSelection: { enabled: true },
  disableRightClick: { enabled: true },
  detectTabSwitch: { enabled: true, severity: 'critical' },
  detectDevTools: { enabled: true, severity: 'critical', action: 'lockout' },
  enforceFullscreen: { enabled: true, reEnterOnExit: true },
  disableKeyboardShortcuts: { enabled: true },
  disableDragDrop: { enabled: true },
  disablePrintScreen: { enabled: true },
}

presets.quiz

Moderate restrictions — for quizzes and practice tests.

Enabled Restrictions

disableCopy, disablePaste, detectTabSwitch, disableRightClick

{
  disableCopy: { enabled: true },
  disablePaste: { enabled: true },
  detectTabSwitch: { enabled: true, severity: 'warning', gracePeriodMs: 2000 },
  disableRightClick: { enabled: true },
}

presets.contentProtection

Prevents content theft — for protected articles, premium content.

Enabled Restrictions

disableCopy, disableCut, disableTextSelection, disableRightClick, disableDragDrop, disablePrintScreen

{
  disableCopy: { enabled: true },
  disableCut: { enabled: true },
  disableTextSelection: { enabled: true },
  disableRightClick: { enabled: true },
  disableDragDrop: { enabled: true },
  disablePrintScreen: { enabled: true },
}

presets.minimal

Lightweight monitoring — just tracks without blocking.

Enabled Restrictions

detectTabSwitch, detectIdle

{
  detectTabSwitch: { enabled: true, severity: 'info' },
  detectIdle: { enabled: true, severity: 'info' },
}

Extending Presets

Spread a preset and add or override specific restrictions:

import { presets } from '@rishikesh7258/react-proctor';

const config = {
  restrictions: {
    ...presets.quiz,
    detectDevTools: { enabled: true },  // add devtools detection
    detectTabSwitch: {
      ...presets.quiz.detectTabSwitch,
      maxViolations: 5,                 // override tab switch limit
    },
  },
};

Or combine multiple presets:

const config = {
  restrictions: {
    ...presets.contentProtection,
    ...presets.minimal,
    // minimal's detectTabSwitch overrides contentProtection's (if any)
  },
};
