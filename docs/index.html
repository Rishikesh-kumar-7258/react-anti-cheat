react-proctor

A configurable, extensible proctoring and integrity enforcement library for React applications with route-level control and a plugin architecture.

Get Started API Reference
npm install @rishikesh7258/react-proctor

Features

12 Built-in Restrictions

Copy/paste, right-click, tab-switch, devtools, fullscreen, keyboard shortcuts, drag-drop, print screen, and idle detection.

Route-Level Control

Different pages get different restriction sets via config or zone components. Router-agnostic.

Plugin Architecture

Register custom restrictions that follow the same interface as built-ins.

TypeScript-First

Full type definitions, autocompletion for config and events. SSR-safe and tree-shakable.

Presets

Pre-configured bundles for exam, quiz, content protection, and minimal monitoring.

Violation Tracking

Every restricted action is captured as a structured event with timestamp, type, severity, and metadata.

Quick Example

import { AntiCheatProvider } from '@rishikesh7258/react-proctor';

const config = {
  restrictions: {
    disableCopy: { enabled: true },
    disableRightClick: { enabled: true },
    detectTabSwitch: { enabled: true, maxViolations: 3 },
    detectDevTools: { enabled: true },
  },
  onViolation: (violation) => {
    console.warn(`Violation: ${violation.type}`, violation);
  },
};

function App() {
  return (
    <AntiCheatProvider config={config}>
      <YourApp />
    </AntiCheatProvider>
  );
}
