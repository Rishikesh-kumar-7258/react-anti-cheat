Getting Started

Install the package and set up proctoring in your React app in under 5 minutes.

Installation

# npm
npm install @rishikesh7258/react-proctor

# yarn
yarn add @rishikesh7258/react-proctor

# pnpm
pnpm add @rishikesh7258/react-proctor

Peer Dependencies

This package requires React 16.8+ (hooks support).

{
  "react": ">=16.8.0",
  "react-dom": ">=16.8.0"
}

The following libraries are bundled — you don't need to install them:

  • visibilityjs — tab visibility detection
  • screenfull — fullscreen API wrapper
  • devtools-detector — devtools detection

Basic Setup

Wrap your application (or a section of it) with AntiCheatProvider and pass a configuration object:

import { AntiCheatProvider } from '@rishikesh7258/react-proctor';

const config = {
  restrictions: {
    disableCopy: { enabled: true },
    disableRightClick: { enabled: true },
    detectTabSwitch: { enabled: true, maxViolations: 3 },
    detectDevTools: { enabled: true },
  },
  onViolation: (violation) => {
    console.warn(`Violation: ${violation.type}`, violation);
  },
};

function App() {
  return (
    <AntiCheatProvider config={config}>
      <YourApp />
    </AntiCheatProvider>
  );
}

Using Presets

For common scenarios, use a preset instead of configuring each restriction manually:

import { AntiCheatProvider, presets } from '@rishikesh7258/react-proctor';

// Use the exam preset (maximum restrictions)
function ExamApp() {
  return (
    <AntiCheatProvider config={{ restrictions: presets.exam }}>
      <ExamContent />
    </AntiCheatProvider>
  );
}

// Or extend a preset
const config = {
  restrictions: {
    ...presets.quiz,
    detectDevTools: { enabled: true },
  },
};

Available presets: exam, quiz, contentProtection, minimal. See Presets for details.

Using the Hook

Access the proctor state and controls from any child component:

import { useAntiCheat } from '@rishikesh7258/react-proctor';

function Dashboard() {
  const { violations, violationCount, isLockedOut } = useAntiCheat();

  return (
    <div>
      <p>Violations: {violationCount}</p>
      {isLockedOut && <p>You have been locked out.</p>}
    </div>
  );
}

Scoped Zones

Apply different restrictions to specific sections of your UI:

import { AntiCheatZone } from '@rishikesh7258/react-proctor';

// Strict zone
<AntiCheatZone
  restrictions={{ disableCopy: { enabled: true }, disablePaste: { enabled: true } }}
  mode="override"
>
  <ExamContent />
</AntiCheatZone>

// No restrictions zone
<AntiCheatZone restrictions={{}} mode="replace">
  <HelpSection />
</AntiCheatZone>

Auto-Lockout

Lock the user out after too many violations:

<AntiCheatProvider
  config={config}
  lockoutAfter={10}
  lockoutComponent={
    <div>
      <h1>Session Terminated</h1>
      <p>Too many violations were detected.</p>
    </div>
  }
>
  <App />
</AntiCheatProvider>
