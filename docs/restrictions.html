Restrictions

All 12 built-in restrictions with their configuration options, default severities, and violation metadata.

disableCopy

Prevents the user from copying content.

OptionTypeDefaultDescription
enabledboolean—Enable this restriction
severitySeverity'warning'Violation severity level
showNotificationbooleantrueShow a toast/message on attempt
notificationMessagestring'Copying is disabled'Custom notification message
allowInputsbooleanfalseAllow copy inside input and textarea

Violation metadata: { attempted: 'copy', target: string }

detectTabSwitch

Detects when the user switches to a different tab or window. Powered by visibilityjs.

OptionTypeDefaultDescription
enabledboolean—Enable this restriction
severitySeverity'critical'Violation severity level
maxViolationsnumberundefinedAuto-lockout after N tab switches
gracePeriodMsnumber0Ignore switches shorter than this (ms)
trackDurationbooleantrueTrack how long the user was away

Violation metadata: { durationMs: number | null, switchCount: number }

detectDevTools

Detects when browser DevTools is opened. Powered by devtools-detector.

OptionTypeDefaultDescription
enabledboolean—Enable this restriction
severitySeverity'critical'Violation severity level
action'warn' | 'redirect' | 'lockout''warn'What happens when devtools is detected
redirectUrlstring—URL to redirect to if action is 'redirect'
checkIntervalMsnumber1000Detection polling interval (ms)

Violation metadata: { devToolsOpen: boolean }

enforceFullscreen

Forces the browser into fullscreen mode and detects exits. Powered by screenfull.

OptionTypeDefaultDescription
enabledboolean—Enable this restriction
severitySeverity'critical'Violation severity level
promptOnMountbooleantrueRequest fullscreen when restriction activates
reEnterOnExitbooleantruePrompt user to re-enter fullscreen on exit
maxExitsnumberundefinedLockout after N fullscreen exits
promptMessagestring—Custom message in re-enter prompt
Note

Browsers require a user gesture to enter fullscreen. When promptOnMount is true, a modal is shown asking the user to click to enter fullscreen.

Violation metadata: { exitCount: number }

disableKeyboardShortcuts

Blocks specific keyboard shortcuts.

OptionTypeDefaultDescription
enabledboolean—Enable this restriction
severitySeverity'warning'Violation severity level
blockedstring[]See belowShortcuts to block
allowListstring[][]Shortcuts to explicitly allow (overrides blocked)
showNotificationbooleantrueShow a toast/message on attempt

Default blocked shortcuts:

Ctrl+C, Ctrl+V, Ctrl+X, Ctrl+A, Ctrl+S,
Ctrl+U (view source), Ctrl+Shift+I (devtools),
Ctrl+Shift+J (console), Ctrl+Shift+C (inspector),
F12, Ctrl+P (print), Ctrl+F (find)

Shortcut format: Modifier keys joined with +. Examples: 'Ctrl+C', 'Ctrl+Shift+I', 'F12'

Violation metadata: { shortcut: string, key: string, modifiers: string[] }

disablePrintScreen

Detects print screen key and overlays the page to prevent screenshots.

OptionTypeDefaultDescription
enabledboolean—Enable this restriction
severitySeverity'critical'Violation severity level
overlayColorstring'#000000'Overlay color when PrintScreen detected
overlayDurationMsnumber1000How long overlay stays visible (ms)
blockPrintDialogbooleantrueIntercept Ctrl+P as well
Limitation

This is a deterrent, not a guarantee. OS-level screenshots cannot be fully prevented by a web application.

Violation metadata: { method: 'printscreen' | 'print-dialog' }

detectIdle

Detects when the user has been idle (no mouse/keyboard activity) for a configurable duration.

OptionTypeDefaultDescription
enabledboolean—Enable this restriction
severitySeverity'warning'Violation severity level
idleThresholdMsnumber60000Time before user is considered idle (ms)
eventsstring[]['mousemove', 'keydown', 'scroll', 'touchstart']DOM events that reset the idle timer

Violation metadata: { idleDurationMs: number }

Severity Levels

SeverityMeaningDefault Use
infoAction logged, no user-facing consequenceText selection, idle detection
warningAction logged, optional user notificationCopy/paste/cut, keyboard shortcuts
criticalAction logged, may trigger lockoutTab switch, devtools, fullscreen exit

Each restriction has a default severity (documented above) that can be overridden in config.
